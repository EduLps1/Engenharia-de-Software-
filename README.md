# 🍽 Restaurante SaaS 

## 1. Descrição do Projeto
Este projeto é um **SaaS (Software as a Service) para gerenciamento de restaurantes**, permitindo que restaurantes gerenciem reservas, cardápio, pedidos e clientes de forma centralizada e online.

### Objetivos
- Automatizar o gerenciamento de reservas e pedidos  
- Facilitar o controle de clientes e histórico de atendimentos  
- Oferecer relatórios sobre vendas, reservas e produtos mais consumidos  

---

## 2. Funcionalidades
- **Gestão de Reservas:** criar, atualizar, cancelar e consultar reservas  
- **Gestão de Cardápio:** adicionar, editar e remover pratos  
- **Gestão de Pedidos:** registro de pedidos realizados no restaurante  
- **Cadastro de Clientes:** informações básicas, histórico de pedidos e reservas  
- **Relatórios e Dashboards:** estatísticas de vendas, reservas e pratos mais populares  
- **Autenticação e Autorização:** controle de acesso de administradores e funcionários  

---

## 3. Tecnologias Utilizadas
- **Backend:** PHP 8.x (Laravel, Slim ou puro PHP)  
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap/Tailwind  
- **Banco de Dados:** MySQL / MariaDB  
- **Versionamento:** Git + GitHub  
- **Deploy:** Servidor Apache/Nginx (opcional Docker)
